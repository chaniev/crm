# Backlog

`backlog/` хранит входящие запросы и задачи на доработку.

- `inbox/` - входящие запросы
- `tasks/` - задачи на доработку
- `done/` - реализованные задачи на доработку

## Команда `зафиксируй`

Если пользователь пишет `зафиксируй`, создай файл `backlog/inbox/YYYY-MM-DD.md` с текущей датой, если его еще нет, и впиши туда все, что написано после первого `зафиксируй`. Если файл уже существует, добавь запись в конец файла.

Опечатку `зафикчируй` считай тем же триггером.
